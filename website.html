What does a game developer do?

Game developer essentially turns ideas into the virtual world by writing codes from various programming languages, find or create sounds/music for your game, make the models for the characters and the environment, add dialogues if there are any, fix bugs to ensure that the game can run properly. Some of the most commonly used coding languages for video games includes: C++, C#, Java(Minecraft!), and a lot of other languages depending on what type of games you want to create and what program you want to use.
